# py-task-cli

Простой консольный менеджер задач на Python для управления вашими ежедневными задачами.

## О проекте

**py-task-cli** - это легковесный инструмент командной строки для управления задачами. Он позволяет создавать, обновлять, удалять задачи, а также отслеживать их статус (todo, in-progress, done). Все задачи сохраняются в локальном JSON-файле.

## Зачем он нужен?

Этот проект решает проблему простого управления задачами без необходимости использования сложных систем. Преимущества:

* Простой и интуитивно понятный интерфейс командной строки
* Быстрое добавление и управление задачами
* Локальное хранение данных без необходимости базы данных
* Возможность фильтрации задач по статусу
* Отлично подходит для личного использования или небольших команд

## Как его использовать?

### Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Установите Poetry:
   ```bash
   pip install poetry
   ```
3. Клонируйте репозиторий или скачайте исходный код
4. Установите зависимости:
   ```bash
   poetry install
   ```

### Использование

Основные команды:

```bash
# Добавить новую задачу
poetry run task-cli add "Купить молоко"

# Показать все задачи
poetry run task-cli list

# Показать задачи с определенным статусом
poetry run task-cli list todo
poetry run task-cli list in-progress
poetry run task-cli list done

# Обновить задачу
poetry run task-cli update 1 "Купить молоко и хлеб"

# Отметить задачу как в процессе
poetry run task-cli mark-in-progress 1

# Отметить задачу как выполненную
poetry run task-cli mark-done 1

# Удалить задачу
poetry run task-cli delete 1
```

### Примеры использования

```bash
# Добавить несколько задач
poetry run task-cli add "Написать отчет"
poetry run task-cli add "Подготовить презентацию"
poetry run task-cli add "Отправить email клиенту"

# Посмотреть все задачи
poetry run task-cli list

# Начать работу над задачей
poetry run task-cli mark-in-progress 1

# Завершить задачу
poetry run task-cli mark-done 1

# Удалить ненужную задачу
poetry run task-cli delete 3
```

## Какую пользу можно извлечь?

* **Упрощение управления задачами**: Быстрое добавление и отслеживание задач без необходимости использования сложных инструментов
* **Повышение продуктивности**: Возможность сосредоточиться на текущих задачах, отфильтровывая выполненные
* **Гибкость**: Простой интерфейс командной строки, который можно интегрировать в ваши скрипты и рабочие процессы
* **Локальное хранение**: Все данные хранятся локально в файле `tasks.json`, что обеспечивает конфиденциальность

### Основные компоненты

* **Task**: Класс, представляющий задачу с атрибутами id, description, status, created_at, updated_at
* **Функции управления задачами**:
  - `load_tasks()`: Загрузка задач из файла
  - `save_tasks()`: Сохранение задач в файл
  - `add_task()`: Добавление новой задачи
  - `update_task()`: Обновление существующей задачи
  - `delete_task()`: Удаление задачи
  - `mark_task_in_progress()`: Изменение статуса на "в процессе"
  - `mark_task_done()`: Изменение статуса на "выполнено"
  - `list_tasks()`: Вывод списка задач с опциональной фильтрацией по статусу
